🎮 LCBD (로그라이크 인디 게임)

진행 기간: 2024.06 ~ 2024.12

내 역할: UI(설정창, 메인화면, 인게임 아이템창, 사운드), 아이템(습득, 사용), 오프닝씬 개발

팀원 수: 10명

이 프로젝트는 Unity 엔진을 활용한 로그라이크 인디 게임 개발로, 창업 동아리 활동의 일환으로 진행되었습니다. 개발 과정에서 일부 기능은 계획대로 완성되지 못했지만, 다양한 UI 요소와 게임 기능들을 구현했습니다.

📌 주요 기능

  • ⚔️ 키 입력 및 마우스 입력으로 몬스터 처치 및 아이템 습득
  • 🎯 스테이지 클리어 방식의 게임 플레이 구조
  • 🛠️ 다양한 UI 요소 (설정창, 키 설정, 사운드 설정)
  • 🎬 오프닝 씬 구현으로 게임의 몰입도 향상
