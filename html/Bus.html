🚌 단거리 버스 노선 논문 포트폴리오

진행 기간: 2022.06 ~ 2022.12

논문 주제: 서울시 청소년들을 위한 교통 데이터 분석 기반의 단거리 버스 노선 선정

팀 인원수: 4명

본 논문은 서울시 청소년들의 통학 편의를 개선하기 위해 교통 데이터 분석을 기반으로 단거리 버스 노선을 선정하는 연구를 진행했습니다.

📌 분석 내용

  • 🚏 버스 이용이 많은 지역 분석
  • 🏫 노원구 내 중고등학교 주변 버스 이용량 분석
  • 🚍 혼잡한 버스 노선의 차내 혼잡도 분석
  • 📊 다익스트라 알고리즘을 활용한 최단거리 노선 추출

📌 사용 기술

  • 💻 사용 언어: Python
  • 📚 라이브러리: numpy, pandas, matplotlib
  • 📈 분석 알고리즘: 다익스트라 알고리즘
