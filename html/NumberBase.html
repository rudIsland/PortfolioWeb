🎮 멀티 숫자야구 게임

진행 기간: 2024.03 ~ 2024.06

내 역할: PM 및 메인 프로그래머

팀원 수: 3명

멀티 숫자야구 게임은 유니티 엔진과 포톤 서버를 활용하여 개발된 멀티플레이 게임입니다.
외부 라이브러리 적용과 서버 통신을 통해 클라이언트 간의 실시간 상호작용을 구현하였습니다.

📌 주요 기능

  • 🔗 닉네임 입력 후 방 생성 및 입장 기능
  • 💬 팀 채팅 기능을 통한 전략적 소통
  • 🎯 팀 간 비밀 숫자 맞추기 게임 진행

🔗 관련 링크

GitHub 링크

📌 개발하며 느낀 점

포톤 라이브러리가 제공해주는 소스를 활용해 제작했기에 기능 구현에는 큰 문제가 없었습니다. 다만, 팀장으로서 부족한 역량으로 역할 분배와 소통에 있어 협업적으로 아쉬웠던 프로젝트였습니다.
