프로젝트 상세 내용

포톤 서버를 활용한 유티니 기반의 멀티 숫자 야구 게임

접속하기

  • 닉네임 지정 후 접속하기를 통해 게임 접속

방 생성 및 입장

  • 방을 생성 하거나 만들어져 있는 방에 입장가능

게임 진행 조건

  • 각 팀의 인원수가 일치해야 하며 방장만이 누를 수 있는 게임시작 버튼을 통해 진행할 수 있습니다.

팀 채팅

  • 팀 기능을 통해 각 팀의 플레이어들 끼리만 채팅이 보이도록 기능 구현

각 팀의 비밀 숫자 정하기

  • 각 팀의 1번이 자신팀의 숫자를 정하고 진행하게 됩니다.

정답 기록 확인

  • 이전에 제시한 번호들의 판정들을 패널을 통해 기록해둠으로써 게임진행 도중에 언제든지 확인 할 수 있도록 구현했습니다.
