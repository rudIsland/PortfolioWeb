프로젝트 정보

  • 카테코리: 게임 개발

  • 게임명: 웹 기반의 벽돌깨기 게임

  • 프로젝트 진행 날짜: 2024.07.12 ~ 2024.07.22

  • Project Git URL: https://github.com/rudIsland/KDT_Project_1

프로젝트 상세 내용

HTML을 기반으로 한 벽돌깨기 게임 제작

스테이지 화면

  • 스테이지를 표시하는 화면이며 Play버튼 클릭시 게임화면으로 넘어갑니다.

게임 화면

  • 3x7 배열의 벽돌과 공, 받침대의 오브젝트로 구성

  • 마우스로 공을 튕길 방향을 선택하고 좌우로 받침대를 움직이며 공을 튕겨내는 방식

아이템

  • 빨강: 현재 생명력을 1 회복시킵니다.

  • 파랑: 공 위치를 초기화 시켜 다시 날려보냅니다.

설정 모달창

  • 속도조절기능: 공의 속도를 조절합니다.

  • 재시작 버튼: 스테이지1 부터 다시 시작합니다.

종료 모달창

  • 점수: 공의 충돌횟수와 남은 생명횟수에 따라 증가한 점수를 표시합니다.

  • 순위: 로컬스토리지에 저장되어있는 점수를 불러와서 내림차순으로 순위를 표시합니다.
